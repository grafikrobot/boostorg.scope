Macro BOOST_SCOPE_FINAL

BOOST_SCOPE_FINAL — The macro creates a uniquely named final scope exit guard.

Synopsis

// In header: <boost/scope/scope_final.hpp>

BOOST_SCOPE_FINAL

Description

The macro should be followed by a function object that should be called on leaving the current scope. Usage example:

BOOST_SCOPE_FINAL []
{
    std::cout << "Hello world!" << std::endl;
};
Note

Using this macro requires C++17.
